视觉定位（visual grounding）关注的是将自然语言表达精确对应到图像中的目标区域，它是多模态模型实现“语言理解落到视觉证据”的关键能力。随着多模态大模型的发展，模型已经能够生成较流畅、较丰富的视觉问答结果，但其回答常常存在“语义上合理、证据上不足”的问题，即用户看到的是答案，却看不到答案究竟建立在哪些图像区域之上。

## 1.2 研究意义

从可解释性角度看，visual grounding 为多模态大模型提供了一种比纯 attention heatmap 更具可验证性的解释接口。它能够将模型输出与图像中的框、掩码或显著区域建立明确对应关系，使解释不再停留在“模型似乎看了哪里”，而是转化为“模型究竟依据了哪块区域”。这种“言之有物”的解释方式有助于提升模型系统的可理解性、可校验性与可信赖性。

从应用角度看，在视觉问答、场景理解、人机交互等任务中，用户不仅关心答案是否正确，也关心答案是否有足够证据支撑。因此，以 visual grounding 为核心研究多模态大模型可解释性，具有明确的方法价值与现实意义。

## 1.3 问题提出

尽管现有多模态大模型已经具备一定的视觉 grounding 能力，但仍存在以下问题：

1. 解释多停留在 attention map 或热力图层面，空间边界不明确，忠实性不足。
2. 现有解释通常只支持“回答到图像”的单向映射，缺少“图像到回答”的反向追踪。
3. grounding 结果通常停留在像素或区域层，缺少概念层抽象，难以形成更自然的人机认知对齐。
4. 可解释信号多数用于事后展示，尚未充分用于错误检测与纠错。

## 1.4 研究内容

围绕上述问题，本文拟开展以下三部分研究：

1. 构建基于 F-LMM 的端到端视觉定位可解释框架，实现从回答到图像、再从图像回到回答的双向 grounding。
2. 将被 grounding 的区域进一步概念化，构建“图像区域-视觉概念-回答片段”三层解释单元，并结合 VCD 与 WordNet 进行知识对齐与补充分析。
3. 基于 grounding、concept 和 attention/region-to-token 等解释信号，设计统计意义上的诊断锚定与纠错机制，用于识别模型对语言先验的过度依赖并辅助修正回答。

## 1.5 论文组织结构

本文后续章节安排如下：第二章综述视觉定位、多模态大模型可解释性、概念层解释与纠错相关研究，并给出本文的问题定义；第三章介绍整体可解释框架与双向 grounding 设计；第四章讨论解释单元的概念化和知识对齐；第五章介绍基于统计诊断锚定的错误检测与纠错方法；第六章给出实验设计与结果分析；第七章总结全文并讨论未来工作。

## 1.6 本章小结

本章从研究背景、研究意义和问题提出出发，说明了 visual grounding 作为多模态大模型可解释接口的必要性，并给出了本文的主要研究内容与整体章节安排。
