# 第二章 相关工作与问题定义

## 2.1 视觉定位研究现状

本节介绍 visual grounding、referring expression comprehension、phrase grounding、generalized grounding 等任务的发展脉络，说明视觉定位已经从经典的目标指代问题逐步扩展为多模态大模型细粒度理解与空间证据建模的基础能力。这里需要重点交代：

- visual grounding 的任务定义和常见输出形式
- 常见数据集与评价指标
- grounding 从传统模型走向 MLLM 的演化趋势

## 2.2 多模态大模型的可解释性研究

本节综述多模态大模型可解释性的典型路线，包括 attention 可视化、gradient 类方法、文本 rationales、区域证据展示和交互式分析工具等。需要指出，现有方法虽然能够提供一定的过程可视化，但在空间实体明确性、因果忠实性和人机可校验性上仍存在明显不足。

## 2.3 attention map 与 grounding 解释的差异

本节是全文的重要铺垫。需要明确说明：attention map 可以作为模型内部行为的观测窗口，但并不天然等同于真实决策证据。相比之下，grounding 的优势在于它提供了更明确的区域边界与更直接的视觉锚点，因此更适合作为“言之有物”的解释接口。这里可以把 attention map 作为对比基线，而不是论文的最终目标。

## 2.4 概念层解释与知识对齐

在像素级证据之外，越来越多研究开始关注将视觉区域映射到概念、属性或语义单元，以提升人类对模型决策过程的理解程度。本节需要综述概念瓶颈、属性解释、视觉概念词表、常识资源对齐等工作，并引出本文将 grounding 区域进一步对齐到 VCD 和 WordNet 的思路。

## 2.5 基于解释信号的错误检测与纠错

本节重点讨论多模态幻觉、视觉证据不足、语言先验过强等问题，以及基于自反思、重采样、外部证据校验和 grounded reasoning 的纠错方法。需要说明：现有研究已经表明，解释信号不应只服务于事后展示，还可以被用来辅助错误检测与性能提升。

## 2.6 问题定义

基于前述综述，本文聚焦如下问题：对于一个多模态大模型给定图像和文本输入后生成的回答，如何建立其输出与图像证据之间的稳定联系；如何将图像证据提升到概念层表达；以及如何利用这种解释链路识别和纠正缺乏视觉证据支撑的错误回答。

进一步地，本文将研究对象拆解为三个层次：

1. 证据层：回答是否能够被 grounding 到可靠的图像区域。
2. 概念层：被 grounding 的区域能否映射到可读的视觉概念。
3. 诊断层：证据与概念信号能否用于错误判定与纠错。

## 2.7 本章小结

本章梳理了 visual grounding、多模态大模型可解释性、概念层解释及纠错相关研究，并据此给出了本文的问题定义，为后续方法章节建立了研究边界。
