一、课题研究的背景和意义（综述国内外相关研究现状，阐述课题的研究目的、意义等）
研究背景：
视觉定位（visual grounding）通常指在给定自然语言表达的条件下，让模型在图像或视频中精确定位与该表达相对应的目标对象或区域，是在视觉场景中“把话指到物”的核心能力。[1] 如图1所示,随着视觉—语言预训练模型和多模态大模型的迅速发展，视觉定位已经从早期的指代表达理解、短语定位等经典任务，扩展到广义视觉定位[2]（可以定位零个、一个或多个目标）、文档图像中文本的视觉定位以及多图像与时空场景下的定位，逐渐成为多模态理解、推理和人机交互的基础问题之一。

图 1Visual Grounding 示意图

然而，现有多模态大模型在开放场景中仍然普遍存在“语义上看似合理但空间上不准确”甚至产生视觉幻觉的问题，缺乏可验证的空间证据和可解释的推理过程，[25]使得围绕视觉定位能力展开的大模型可解释性研究具有突出的理论意义和现实紧迫性。
从更系统的视角来看，如图2所示，在“人-机”[4]层面，visual grounding 则为人类认知与多模态模型的机器认知之间建立共享的感知与语义参照系：模型不仅输出答案，还需要通过框选、掩码或高亮区域的形式显式展示其决策所依托的视觉证据，使人类可以在同一图像空间中理解和审视模型的判断，从而形成“认知同调”，提高用户对模型决策过程的可感知性与可校验性。 随着人本评测与可解释视觉定位研究的发展，这种基于 visual grounding 的人-机共识被越来越多地视为多模态大模型在复杂场景分析任务中实现可控性和可信赖性的前提条件，也为本课题从视觉定位角度切入大模型可解释性研究提供了直接的现实动因。

图 2 人-机认知对齐示意图
国内外研究现状：
Xiao 等[3]系统梳理了过去十余年视觉定位领域的发展，将视觉定位（也称指代表达理解、短语定位）界定为在图像中根据自然语言表达定位一个或多个目标区域，并从完全监督、弱监督、半监督、无监督、零样本、多任务以及广义视觉定位等多种设定出发，综述了相关数据集与评测指标，指出视觉定位在视觉导航、人机对话、视觉问答等应用中的重要作用。 与此同时，国内已有综述从多模态预训练到多模态大模型的发展脉络出发，对相关模型的架构、训练方式、评测体系与演进趋势进行了系统梳理，说明视觉定位能力已越来越多地被放入统一多模态模型能力框架中加以理解与讨论。[24]

图 3 Visual Grounding的发展
如图3所示，自 2021 年以来，随 grounded pre-training、grounding 多模态大模型、giga-pixel grounding 等新概念的提出，视觉定位已从单一任务演变为支撑多模态对话和复杂推理的底层能力，对标准化任务设定和评价体系提出了新的挑战。
Pantazopoulos 等[1]中进一步从现代通用视觉—语言模型（VLM）的角度，总结了具备视觉定位能力的 VLM 的核心组成、训练范式与典型应用，指出显式的视觉定位既是实现细粒度下游任务（如精细问答、可解释图像描述、具身控制）的前提，也是建立“模型预测—实际视觉区域”之间可解释联系的重要手段，可有效降低语义正确但空间错误的回答风险。
在多模态大模型层面，一类代表性工作(Shikra[5],Kosmos-2[6],Ferret[7])通过将带空间标注的数据集（如 Visual Genome[8]、RefCOCO 系列[9][10]）转化为指令格式，对多模态大模型进行指令微调，从而赋予模型区域级理解与对话能力，相关进展被 LION[11] 模型总结为利用视觉定位任务向大模型注入细粒度、空间感知的视觉知识。 也有工作系统性研究视觉定位在多模态大模型中的设计空间，例如[12]基于 LLaVA-1.5 比较不同的视觉定位范式和数据构造方式，发现合适的范式与数据设计可以在 RefCOCO[9]/RefCOCO+[9]/RefCOCOg[10] 上显著提升定位精度，为后续方法提供了经验性指导。
在模型结构与能力平衡方面，Wu 等[13]提出的 F-LMM 通过“冻结”现有多模态大模型，仅在其内部注意力的词—像素对应关系之上叠加少量可训练的卷积掩码头，并结合 SAM 掩码精修模块，将注意力图转化为分割掩码，实现短语分割和叙述性全景定位等任务，在保持原有对话与知识问答能力基本不受损的情况下，在多种定位基准上取得了有竞争力的性能。 另一些工作（如 VGR: Visual Grounded Reasoning[14]）提出在多模态链式思维推理中显式输出定位框与推理步骤，使模型在回答复杂视觉问题时主动检索关键区域，从而同时提升推理准确性和过程的可解释性。 近期针对多模态大模型内部机制的分析工作[15]也表明，通过探测不同层的 token 表征，可以更细致地理解模型在视觉问答任务中如何逐步利用视觉和文本信息，这为从表示层面研究视觉定位与推理的可解释性提供了方法学基础。
研究意义
本课题的研究意义在于：以visual grounding为核心抓手，把多模态大模型的输出与可验证的空间证据（框/掩码/关键区域）建立明确对应关系，从而提升模型在场景理解与视觉问答中的可解释性，减少“答案看似合理但缺乏证据或证据不一致”的黑箱问题。 同时，grounding 通过把语言表达锚定到人类可直接感知的图像区域，形成人机共享的参照系，促进人机认知对齐：人可以直观看到模型“在看哪里、依据什么”，进而更容易校验、纠错与形成共识，提升复杂场景分析任务中系统的可控性与可信赖性。


二、课题研究内容、可行性分析及拟解决的关键问题和难点（可说明是否具有前沿性和创新性）
研究内容
本课题拟围绕“聚焦视觉定位（visual grounding）的多模态大模型可解释性”开展系统研究，整体思路是：以视觉定位为核心接口，让模型的每一次视觉决策都“言之有物”，即都能在图像中给出可验证的空间证据，并进一步利用这些证据来解释和纠正模型的行为。 具体研究内容包括：
1.端到端的 visual grounding 可解释框架
  如图1所示，在现有“回答→图像”单向 grounding 基础上，构建“回答→图像”“图像→回答”双向的端到端可解释系统：一方面，对多模态大模型的回答进行区域级 grounding，使用户能看到“答案对应图像的哪里”；另一方面，从图像中被高频关注或被错误利用的区域出发，反向追踪其在生成回答过程中的作用，实现从图像证据到语言输出的因果链条展示。 同时，对比仅基于 attention map 的粗粒度热力图解释，强调 grounding 必须输出明确的框/掩码等空间实体，使解释真正具备“言之有物”的证据支撑。

图 1 端到端的 visual grounding 可解释框架
2.解释单元的概念化与知识对齐
在获得区域级 grounding 的基础上，引入“解释单元（explanatory units）”概念，将图像中被 ground 的区域进一步对齐到视觉概念库。并且，国内关于多模态可解释学习的研究也表明，将视觉证据进一步映射到属性或概念层表示，有助于提升模型决策过程的可理解性，因此将 grounding 结果与概念知识进行对齐具有明确的方法学依据。[26]如图2所示，形成“图像区域–中层概念–自然语言描述”三层对应关系。 通过构建这一概念空间，可以将原本只在像素层面的 grounding 上升到概念层面的可解释：既指出“模型看的是哪一块图像”，也能用人类熟悉的概念和词汇解释“它把这块看成了什么、为什么这样回答”，从而更好地实现人机认知对齐。

图 2解释单元的概念化与知识对齐
3.基于 grounding 的错误检测与纠错机制
  利用模型内部 attention / hidden states 与外部 grounding 结果之间的一致性与冲突关系，设计“可解释信号驱动的纠错”机制：当检测到模型在回答中大量依赖自身历史文本或语言先验，而 grounding 证明图像中缺乏对应证据时，将其判定为潜在的视觉幻觉或错误答案，再通过多轮自我反思或重采样进行纠错。 该部分借鉴近期“将多模态 grounding 视为纠错透镜”的研究思路[16]，将 visual grounding 与 textual rationale 作为校准 MLLM 输出的双重证据，从而探索“让可解释性不仅用于事后说明，还能反向提升模型最终性能”的新范式
可行性分析
在数据与模型基础上，本课题具有良好的实现条件：一方面，现有 Visual Genome[8]、RefCOCO 系列[9][10]等公开数据集提供了丰富的区域标注、grounding 信息以及带解释的 VQA 样本，可为视觉定位与可解释学习提供监督基础。 另一方面，Deepseek-VL2[17]、Qwen-VL [18] 等开源多模态大模型及多种开源 grounding 模型[19][20][21]已经广泛可用，便于在统一框架下进行端到端微调与探针分析，为 probing 不同层的 token 表征和 attention 模式提供了成熟的工程支撑。 此外，已有工作表明，通过把视觉 grounding 作为额外监督，可以同时提升定位性能和 VQA 准确率，甚至将 grounding 作为“纠错透镜”来缓解幻觉问题[16]，这为本课题提出的“解释-驱动纠错”路径提供了直接的可行性依据。
拟解决的关键问题和难点
1.解释的可信度与忠实性问题
  现有 attention map 往往缺乏因果意义，容易出现“看对地方但答错题”或“看错地方却答对题”的情况，本课题需要设计兼顾空间精度与因果性的评价指标与训练策略，确保 grounding 与解释真正反映模型内部决策依据，而非仅仅是事后可视化。
2.双向 grounding 与粒度对齐难题
  将“回答→图像”的 grounding 扩展到“图像→回答”的反向解释，需要在 token 粒度、区域粒度与概念粒度之间建立稳定映射，解决一对多、多对一以及跨层抽象粒度不一致等问题，这是构建端到端可解释链条的核心技术难点之一。
3.基于可解释信号的安全纠错策略
  如何在不显著损伤原有正确回答的前提下，可靠地利用 grounding 与 attention 中的不一致作为错误信号进行干预与纠错，需要在错误检测阈值、干预方式（重推理、重采样、拒答等）以及多源冲突信息融合上进行系统设计。 同时，还需防止模型过度依赖语言先验或过度“迷信”视觉证据，导致新的偏差与失败模式出现。
前沿性和创新点
本课题的前沿性在于，顺应多模态大模型和视觉定位融合的最新趋势，以 visual grounding 为纽带，将“可解释性、视觉证据、人机认知对齐与性能提升”统一在同一研究框架之下。 相比传统仅依赖 attention 热力图的解释方法，本课题的创新主要体现在：
1.提出“回答→图像”与“图像→回答”双向 grounding 的端到端可解释系统，使解释从单向对齐提升为完整的因果链条展示；
2.将被 ground 的图像区域概念化为“解释单元”，并对齐到视觉概念库与 WordNet 等知识资源，实现从像素级证据到概念级认知的跨层对齐，强化人机认知对齐；
3.探索以 grounding 等可解释信号为驱动的纠错与性能提升机制，把可解释性从“事后说明”提升为“在线校准与纠偏”的重要手段，为降低多模态大模型幻觉、提升可靠性提供新思路。
三、研究方法或技术路线(重点论述技术方案)
本课题拟构建一个“以 F-LMM 为底座、结合双向 grounding 与概念知识和内部表征分析的端到端可解释框架”。如图1所示，技术路线大致包括三层：
第一层，在视觉定位能力方面，采用 F-LMM [13]作为基础 grounding 模块，对选定的多模态大模型进行“冻结主体 + 轻量掩码头”的改造，利用其内部注意力中的 word-pixel 对齐关系，通过少量可训练的 CNN 掩码头与 SAM 掩码精修模块，将文本短语映射为高质量的 bbox/seg 掩码，实现在 RefCOCO 系列[9][10]、叙述性全景定位[22]等任务上的精细化视觉 grounding，同时保持原有对话与指令跟随能力不受明显损伤。
第二层，在可解释系统层面，借鉴 VL-InterpreT [23]的设计思想，开发面向多模态大模型的双向 grounding 交互式可视化工具：一方面，从回答到图像方向，将生成的答案分解为关键 token / phrase，并通过 F-LMM 输出对应的 bbox/seg，展示“模型回答依托的空间证据”；另一方面，从图像到回答方向，沿着 transformer 各层的注意力与隐状态，将被关注的区域追踪到最终输出 token，帮助分析“特定区域如何影响生成的语言”。 同时，将这些被 ground 的区域与 VCD 等视觉常识资源对齐，把每个区域映射为若干视觉概念和属性，构造“图像区域–视觉概念–自然语言解释”的多层“解释单元”，实现从像素证据到概念层可解释知识的统一表示。
第三层，在纠错与性能提升方面，结合 Yu & Lee 提出的“通过探测不同层 token 表征刻画模型由视觉定位→任务推理→答案解码的阶段性信息流”的分析框架，对多模态大模型在 VQA / 场景问答任务中的内部表征进行 probing：当检测到模型回答主要依赖自身历史文本或高层语言先验，而 grounding 与中层视觉 token 表征显示图像中缺乏相应证据时，将其视为潜在幻觉或错误，通过设计基于可解释信号的再推理、重采样或提示重构策略，对输出进行有针对性的纠错和校准，使 visual grounding 不仅用于“事后解释”，还能实质性提升模型在复杂场景分析任务中的鲁棒性与整体性能。
